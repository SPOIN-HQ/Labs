A public space for code, ideas, and findings worth sharing.  
Algorithms, prototypes, research notes — anything we've built that might help someone else build something better.

### What lives here

The repo is loosely organized by topic. Each folder is self-contained with its own notes.

```labs/
├── algorithms/      # Algorithm implementations and notes
├── experiments/     # Prototypes and proofs of concept
├── research/        # Write-ups, papers, and exploratory notes
└── utils/           # Small utilities worth reusing

This structure is a starting point — folders come and go as the lab grows.

### Getting started

```bashgit clone https://github.com/SPOIN-HQ/Labs.git
cd Labs

Most folders are independent. Open the one you're interested in and check its local README for setup details.

### Contributing

Contributions are welcome — sharing is the whole point of this repo.

- 🐛 **Found a bug?** Open an [issue](https://github.com/SPOIN-HQ/Labs/issues).
- 💡 **Got an improvement?** Send a pull request.
- 🤔 **Have an idea?** Start a [discussion](https://github.com/SPOIN-HQ/Labs/discussions) or open an issue first.

No formal style guide. Write code you'd be proud to read a year from now, document your reasoning, and we'll figure out the rest together.
